Краткий тест по работе с Makefile

1. Какая из команд, приведенных ниже, компилирует файл в объектный код?
 - vim main.c
 - touch hello.h
 - gcc –o hello.o –c hello.c

2. Как узнать информацию о той или иной команде и как с ней работать (как запустить справочную страницу)?
 - gcc --version
 - man название_команды
 - wq

3. Как правильно закрывать текстовый редактор?
 - нажать «esc» и написать «:wq»
 - нажать на крестик
 - нажать «Backspace»

4. Какая команда собирает не исполняемый файл, а динамическую библиотеку?
 - gcc -o libHello.so -shared -fPIC hello.c
 - gcc -o hello hello.o main.o
 - gcc -o hello main.c

5. Как запускается простой исполняемый файл?
 - ' :wq '
 - ' ./ '
 - ' -c '
